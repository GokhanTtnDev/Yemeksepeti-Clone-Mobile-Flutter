## ✨ Features & Highlights
* **Advanced Animations:** Includes complex scroll animations and coordinate-based item removal.
* **Custom Sliver App Bar:** Features a smooth, custom sliver app bar with animated color transitions that match modern food delivery app standards.
* **Modern UI/UX Design:** Pixel-perfect screens with clean layouts, focusing on high user engagement.
* **Fully Responsive:** Adapts beautifully to different mobile screen sizes and orientations (iOS & Android).
* **Clean Code Architecture:** Code is modular, easy to read, structured for learning, and completely human-written.
